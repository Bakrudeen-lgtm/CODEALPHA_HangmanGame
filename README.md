# Play Hangman from command-line interface

In this project, we make the game Hangman playable from [command-line interface](https://en.wikipedia.org/wiki/Command-line_interface).

Concepts Used 

 1.Lists
 
 2.Loops (while, for)
 
 3.Condition statements (if)
 
 4.Strings
 
 5.User input
 
 6.Random module
 
 7.Possible Improvements
 

You can improve the game by adding:

1.Hint system

2.Difficulty levels

3.ASCII hangman drawing

4.Score tracking

5.Multiplayer mode


For example, the classic hangman stages:

  O
 /|\
 / \

can be displayed after each wrong guess.
